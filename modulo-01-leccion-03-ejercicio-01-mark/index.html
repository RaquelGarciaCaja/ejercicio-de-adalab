MARCADO DE PALABRAS

    Habría que tener en cuenta lo siguiente:
  • El elemento mark representa en un texto las palabras más importantes marcado o resaltado como referencia o anotación, debido a su relevancia o importancia en un contexto particular.
  • Por otro lado, mark indica una parte del contenido del documento que probablemente sea relevante para el usuario. Por ejemplo, se puede utilizar en una página que muestra los resultados de búsqueda para mark resaltar cada uno de estos por palabra.
  • Para finalizar habría que tener en cuenta que no se debe usar el elemento mark para resaltado de sintaxis; usa el elemento span para este fin.
